# guldgruvan
website for the preschool Guldgruvan.

## Getting started

You'll need `nodejs` and `yarn` installed.

Start of with
```
yarn
```

Here's a couple of available tasks:
```
yarn develop
yarn build
yarn serve
yarn test
yarn lint
```
